# Dentally MCP Server
Connects Claude to your Dentally practice data.

## Deploy to Render.com
1. Connect this repo to render.com
2. Build: `npm install` Start: `npm start`
3. Add env var: DENTALLY_API_TOKEN

- "How much have we invoiced vs received this month?"
- "Compare this month revenue to last month"
